# Zomato Analysis (Interactive Dashboard creation using Excel, PowerBI & Tableau)

## Project Objective
To analyze restaurant data and provide actionable insights into cuisine distribution, service availability, and customer ratings to support better business and customer decisions.

## Dataset used
https://github.com/Shruti-1299/Zomato-Project/blob/main/Zomato%20data.xlsx

## Questions (KPIs)
What is the average cost for two people across restaurants?

What is the overall average rating of restaurants?

Which country offers the highest number of cuisines?

What are the top 5 countries with the most diverse cuisines?

How does cuisine variety differ across countries?

How has the number of restaurants grown over the years?

Which year saw the highest number of restaurant openings?

Are there any noticeable trends or fluctuations in growth?

What percentage of restaurants offer online delivery?

How many restaurants provide table booking facilities?

Is there a relationship between service availability and ratings?

Which cities have the highest number of restaurants?

How does cost for two vary across different cities or countries?

##  Project Process
Collected Zomato dataset 

Cleaned data (removed duplicates, handled missing values)

Prepared data (created fields like cost, rating, total restaurants)

Built dashboard in Excel,PowerBI & Tableau

Added charts (bar, line, donut, gauge)

Applied filters (year, country, rating, delivery)

Analyzed data to find trends and insights
